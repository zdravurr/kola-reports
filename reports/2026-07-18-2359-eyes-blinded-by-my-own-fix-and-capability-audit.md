## 1. 🔴 КОРЕНЬ СЛЕПОТЫ — МОЯ СОБСТВЕННАЯ РЕГРЕССИЯ

Вчера я добавил «докажи свежесть кадра»: `rm -f /sdcard/ui.xml` ПЕРЕД съёмкой.
Это ОСЛЕПИЛО телефон целиком.

Механика:
- ssh-сессия работает под **u0_a313**;
- `uiautomator dump` идёт под **su** и создаёт файл владельцем **u0_a304**, права **660**;
- пока файл СУЩЕСТВОВАЛ, дамп перезаписывал СОДЕРЖИМОЕ, сохраняя прежнего владельца → `cat` читал;
- после `rm` дамп создавал файл ЗАНОВО с чужим владельцем → `cat` = **Permission denied**
  → пусто → «СНИМОК НЕ СДЕЛАН» на КАЖДОМ кадре.

**Лечили чтение УСТАРЕВШЕГО — сломали чтение ВООБЩЕ.** Второй отказ хуже первого: слепой
воркер выглядит работающим.

### Диагноз ПО ЗАМЕРУ, а не по подозрению
Версия «забита память» (самая частая причина) — **ОТВЕРГНУТА фактом**:
`/sdcard` занят **23%**, свободно **85 ГБ**; прямой `adb shell uiautomator dump` отдал **37 КБ**,
`screencap` — **507 КБ**; adb online, ssh-мультиплексор тёплый. Устройство было ЗДОРОВО.

### Починка — КОНСТРУКТИВНАЯ, а не порядком команд
- **уникальное имя файла на каждый кадр** → унаследовать прошлый снимок НЕВОЗМОЖНО в принципе;
- **rm/dump/cat/уборка одной командой ПОД ROOT** → права не мешают читать;
- файл удаляется сразу → `/sdcard` не копит кадры (гигиена встроена).

ДОКАЗАНО: два кадра по 20471 байт, осиротевших файлов на устройстве — **0**. Коммит `05ef0ec`.

## 2. ГЛАЗА НЕ ИМЕЮТ ПРАВА УМИРАТЬ МОЛЧА (`lib/eyes_health.py`)

Лестница самолечения, проверка ПО ФАКТУ после КАЖДОЙ ступени:
`retry → relink → hard_relink → чистка памяти → пробуждение экрана`.

**ДОКАЗАНО В ОБЕ СТОРОНЫ (живьём):**
- имитация слепоты → поймано; retry и relink не помогли; **ВЫЛЕЧЕНО ступенью `hard_relink`**;
- неустранимая слепота → пройдена вся лестница → **FAIL-CLOSED: отказ работать**, успех НЕ доложен.

Врезано туда, где слепота дороже всего:
- ответ покупателю без зрения не отправляется;
- **`lead_guarantee` без зрения НЕ ТРАТИТ НАЗНАЧЕНИЯ** — бюджет попыток обязан уходить на
  работу, а не на слепые побудки, приближающие «эскалацию» по лиду, с которым всё в порядке,
  кроме наших глаз.
