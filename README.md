# kola-reports

**Reports only.** Plain-text markdown, readable directly by URL.

No code. No config. No secrets. No phone numbers. No buyer or owner personal data.
Every file passes a fail-closed secret/PII scan before it is pushed here.

- `reports/YYYY-MM-DD-HHMM-topic.md` — one report per file.
- Read raw: `https://raw.githubusercontent.com/zdravurr/kola-reports/main/reports/<file>.md`
